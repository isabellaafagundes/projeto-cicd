# Projeto CI/CD

Projeto desenvolvido para a disciplina de Engenharia de Software.

## 📌 Objetivo
Implementar um pipeline CI/CD com testes automatizados.

## 🚀 Tecnologias
- Python
- Pytest
- GitHub Actions

## 🧪 Testes
- 20 cenários de teste (normais e de erro)

## ⚙️ Pipeline
O pipeline executa:
- Testes automatizados
- Build
- Deploy
- Notificação

## ▶️ Como rodar localmente
pip install -r requirements.txt
PYTHONPATH=. pytest

## 📦 CI/CD
Executado automaticamente via GitHub Actions.
